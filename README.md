Project Overview
This project simulates a real-world logistics and supply chain environment for a mid-size freight company using pure SQL on SQLite.
It models customers, warehouses, carriers, orders, and shipments, and uses SQL queries to answer practical business questions about delivery performance, warehouse utilization, and customer demand.

The goal is to demonstrate how SQL alone can be used to build an analytics layer on top of operational logistics data—similar to what a Data/BI/Analytics Engineer would do in a logistics firm.

Business Context
A logistics company wants to:

Track on-time vs delayed deliveries by carrier

Understand which warehouses handle most volume

Identify key customers and regions driving revenue

Monitor freight cost patterns across routes and carriers

This project shows how these questions can be answered with well-structured tables and analytical SQL queries.
